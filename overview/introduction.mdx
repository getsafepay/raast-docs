Safepay’s Raast APIs are designed to help aggregators, merchants, and fintech platforms accept and process real-time payments using Pakistan’s Raast instant payment network.

This documentation covers:
- How to get started with a Safepay Aggregator account
- Requesting and using access keys
- Creating merchants and initiating payments
- Generating and using QR codes (static and dynamic)
- Handling refunds and best practices

Use this documentation as a step-by-step guide to implement the full payment lifecycle with Safepay and Raast.
